# NeoRank - Next-Gen Gaming Bio-Link & Stats Platform

NeoRank is a futuristic landing page for a "Living Gaming Profile" platform. It moves beyond static link pages by providing real-time game stats, API integrations, and a high-tech visual identity designed for the modern gamer.

## 🚀 Tech Stack

- **Framework:** Next.js 14+ (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **3D Graphics:** Three.js + React Three Fiber
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Design:** Glassmorphism + Bento Grid Layout

## ✨ Key Features

- **Futuristic 3D Hero:** Interactive floating 3D elements powered by React Three Fiber.
- **Live Stats Mockup:** High-fidelity "LoL Card" showing champion mastery, ranks, and live win rates.
- **Bento Feature Grid:** Clean, organized sections highlighting the platform's core capabilities.
- **S-Tier UI:** Premium glassmorphic effects, blur filters, and a high-contrast palette (Deep Charcoal, Electric Lime, Cyber Magenta).
- **Responsive Design:** Fully optimized for desktop and mobile browsers.

## 🛠️ Getting Started

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open in browser:**
   Navigate to [http://localhost:3000](http://localhost:3000)
